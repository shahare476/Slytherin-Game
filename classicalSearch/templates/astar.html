Overview of Implemntaion

  1. It has 8X8 grid, hence matrix of 8 rows and 8 column holds food location and snake locations
  2. Starting from snake head, food position eculidian distance is calulated as heuristic.
  3. A priority queue is maintained based on heuristic, each item in the queue contains curent node, actual distance till now, heuristic and node via which it came.
  4. Also note that only valid node is taken, now node related to min. heuristic node in priority queue is searched.
  5. When food is reached, algorithm back-track the nodes visited to reach here. Now it is converted into list of direction.
  6. These list of directions along with snake positions is appended into list, until snake do not find a valid path to food, this is when the algorithm stops.
