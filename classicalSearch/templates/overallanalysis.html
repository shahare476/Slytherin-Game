Overall Analysis

  1. Most time is taken by Hamiltonian path, as we perform one bfs to find a path between head and tail, then converting that path to hamiltonian path takes exponential time complexity, hence the highest.
  2. Least time is consumed by hamiltonian cycle and dfs, hamiltonian cycle consumes least time because we find cycle for only one iteration, and after that for next subsequent iteration linear time is consumed.
  3. DFS performs worst in average score of the snake.
  4. BFS and Astar is nearly same as both algorithm have some similarity between search patterns.
  5. Hamiltonian path is significantly larger score than other two, as it tries to visit all vertex hence minimize chance of not finding path.
  6. Hamiltonian gives perfect score because it always tries to visit all nodes and hence will have score equal to all node.
