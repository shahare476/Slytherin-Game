Overview of Implemntaion

  1. It has 8X8 grid, hence matrix of 8 rows and 8 column holds food location and snake locations
  2. Starting from inital snake position(i.e at 0,0), a path back to tail is generated.
  3. two adjecent node are expanded so that path returning to the snake is increased.
  4. The above step is done until all vertex is covered once(hamiltonian cycle)
  5. (Note: snake will cover the board only if it can cover all vertex, therefore snake has to trace the same hamiltonian cycle again and again at least when it is long enough)
