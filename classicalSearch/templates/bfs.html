Overview of Implemntaion

  1. It has 8X8 grid, hence matrix of 8 rows and 8 column holds food location and snake locations
  2. Starting from snake head, food is searched in bfs manner i.e using queue (i-1,j-1,i+1,j+1) given all this is valid location in matrix to go.
  3. A valid location is one which do not contains snake body or is not a bounday
  4. When food is reached, algorithm back-track the nodes visited to reach here. Now it is converted into list of direction.
  5. These list of directions along with snake positions is appended into list, until snake do not find a valid path to food, this is when the algorithm stops.
